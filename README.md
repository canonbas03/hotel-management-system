# Hotel Management System

A **full-stack hotel management system** built with C#, ASP.NET MVC, SQL Server, Entity Framework, and SignalR.  
Designed to manage multiple staff roles, streamline check-in/check-out, track room status, process orders, and handle administrative tasks. This project demonstrates **full-stack web development, real-time updates, and database-driven applications**.

![Landing Page](https://github.com/user-attachments/assets/9edc2f07-9985-47a7-91cf-3029f75ee6c1)

---

## Key Highlights

- Full-stack system supporting multiple staff roles: reception, housekeeping, maintenance, bar service, and administration.  
- Real-time updates using **SignalR** to notify staff about new orders, requests, and room status changes.  
- SQL Server database designed with **Entity Framework** migrations and relational modeling.  
- Dynamic dashboard for monitoring room status, managing guests, and processing orders.  
- Implements role-based access control for secure staff operations.  
- Demonstrates **C#, ASP.NET MVC, SQL Server, AJAX, and frontend skills**.

---

## Features

- **Landing Page** – Welcoming interface for guests and staff login.  
- **Reception Module** – Check-in/check-out, room allocation, and guest management.  
- **Housekeeping Module** – Track room status and manage cleaning schedules.  
- **Bar Service Module** – Manage orders and track payments in real time.  
- **Administration Module** – Staff management, reports, and audit logs.  
- **Real-Time Notifications** – SignalR updates for all staff roles.  

---

## Technologies Used

- **Backend:** C#, ASP.NET MVC, Entity Framework, SignalR  
- **Frontend:** HTML, CSS, JavaScript, AJAX  
- **Database:** SQL Server, relational tables for rooms, guests, staff, and orders  
- **Version Control:** Git/GitHub
